【概要】
このTo Do Listテンプレートは、
スマホでも見やすく快適に使える、
「やること」「買うもの」がパッとわかるシンプル設計。
温かみのある優しいデザインで、毎日のタスク管理に役立ちます。

📝 デモ版ではローカル保存（ブラウザ内）で動作しますが、
完全な保存・管理機能は完全版でお試しください。

【このデモ版で体験できること】
レスポンシブ対応（スマホ・タブレットでも使いやすい）

チェックボックスで完了管理が可能

タイトル（見出し）は編集可能

項目の追加・削除が可能（ページを閉じると内容は消えることがあります）

【ファイル構成】
bash
コードをコピーする
index.html        ... メインのHTMLファイル  
assets/style.css  ... レイアウトと配色を管理するCSSファイル  
js/javascript.js  ... 動作制御（追加・削除など）のJavaScriptファイル  
【注意事項】
このデモ版では、ブラウザのlocalStorageを使用して一時的にデータを保存しています。

永続的な保存や複数リストの切り替え機能は含まれていません。

SEOタグや解析タグ（Google Analyticsなど）は含まれていません。

ソースコードはご自身の学習や確認用途としてご自由にご覧いただけます。

【動作確認について】
このリポジトリをクローンまたはZIPでダウンロード

index.html をブラウザで開いて確認
